# 博客文章创建和发布指南

## 方式一：使用 Hugo 命令创建（推荐）

### 1. 创建中文文章

```bash
# 创建一篇新的中文博客文章
hugo new blog/zh/文章名称.md

# 例如：
hugo new blog/zh/my-new-post.md
```

### 2. 创建英文文章

```bash
# 创建一篇新的英文博客文章
hugo new blog/en/文章名称.md

# 例如：
hugo new blog/en/my-new-post.md
```

## 方式二：手动创建文件

直接在 `content/blog/zh/` 目录下创建 `.md` 文件，使用以下模板：

```markdown
---
title: 文章标题
date: 2025-01-01T12:00:00+08:00
tags: ["标签1", "标签2"]
featured: true
---

文章摘要部分（会显示在列表中）

<!--more-->

这里是正文内容...
```

### Front Matter 字段说明：
- `title`: 文章标题
- `date`: 发布日期（格式：YYYY-MM-DDTHH:mm:ss+08:00）
- `tags`: 标签列表
- `featured`: 是否精选（true/false）
- `series`: 系列文章（可选）

## 本地预览

```bash
# 启动本地服务器预览
hugo server -D

# 然后在浏览器打开 http://localhost:1313/
```

## 发布到 GitHub

### 1. 检查状态

```bash
git status
```

### 2. 添加文件

```bash
# 添加单个文件
git add content/blog/zh/你的文章.md

# 或者添加所有更改
git add .
```

### 3. 提交更改

```bash
git commit -m "新增文章：文章标题"
```

### 4. 推送到远程仓库

```bash
git push origin master
```

## 完整示例流程

```bash
# 1. 创建新文章
hugo new blog/zh/my-awesome-post.md

# 2. 编辑文章内容（使用你喜欢的编辑器）

# 3. 本地预览
hugo server -D

# 4. 确认无误后，添加并提交
git add content/blog/zh/my-awesome-post.md
git commit -m "新增文章：我的优秀文章"

# 5. 推送到 GitHub
git push origin master
```

## 注意事项

1. **文件名建议**：使用英文和连字符，例如 `my-new-post.md`
2. **日期格式**：确保日期格式正确，时区使用 `+08:00`（中国时区）
3. **标签**：使用数组格式，例如 `["工作", "生活"]`
4. **文章摘要**：`<!--more-->` 之前的内容会作为摘要显示在列表中
