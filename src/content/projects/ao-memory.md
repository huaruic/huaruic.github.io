ao-memory 探索如何把 AI 记忆从单一应用中解耦出来，在 AO 生态中保存、分享和组合上下文，让用户更直接地拥有自己的记忆资产。

项目将去中心化存储、AO Process 与 AI 对话连接在一起，是对可携带 Agent Memory 的一次原型验证。

## 技术

React、TypeScript、Vite、AO Network、aoconnect、Arweave Wallet 与 Lua。
